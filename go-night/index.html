About the Plugin

Go Night Dark Mode/Night Mode WordPress Plugin enables dark mode or night mode with multiple options for your website. Dark mode is a new trend on web design. Desktop apps like Adobe Softwares have already implemented dark mode into their UI, now this plugin adds dark mode/night mode feature into your WordPress installation.

Plugin Options:

Detect OS Color Scheme
Set dark mode based on system configurations.
Automatic Mode
Automatically turn dark mode on certain hours.
Include in Specific Pages
Include the button only on specific pages on your site.
Button Type
Choose the type of the toggle button.
Button Size
Set the size of the button.
Button Text
Set the label of the button.
Button Margin
Set the margin (space around) the button.
Button Location
Choose the location of the button on the page.
Include in All Pages
Add the button to the all pages on your website.
Save Last Time Mode
Set the mode that user selected the last time he/she visited your site.
Invert Button
Add an extra button (invert button) near your toggle button. Invert button inverts any color!
Shortcode
Add the button to the location you want via inserting the shortcode.
Set Colors
Set dark mode colors for backgrounds, text, input, buttons, links etc.
Dark Mode Logo
Define a light version of your logo to be shown on dark mode.
Custom Elements
Choose the areas of your website that should stay the same in dark and light mode.
