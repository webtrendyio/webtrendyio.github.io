Coronavirus Safety Message

Show vital health instructions to be safe against Corona virus (COVID-19)

About the plugin

Coronavirus Safety Message is a WordPress plugin that gives safety instructions against Corona virus (COVID-19). The instructions and information are derived from World's Major Health Organizations like WHO.int (World Health Organization). The plugin is very lightweight and small in size. We will reqularly keep updating the plugin with the latest information available. The plugin is compatible with the latest version of WordPress (5.3) and will run cross-platform, cross-browser.

How it works?

01
Install the plugin

After downloading the plugin, go to your WordPress dashboard, install and activate the plugin.

02
Refresh your site

Now go to front end of your website and refresh the page to see Corona Safety Messages.

Why use it?

Raising awareness on Corona virus (COVID-19) and helping other human kind get to know the virus better, is the responsibility of all of us. This little plugin gives health instruction and information to help prevent the spread of the virus. We hope it may help saving lives.
